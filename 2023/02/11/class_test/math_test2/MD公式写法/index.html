上下标

上标表示：^

下标表示：_

  • 默认情况 ：上下标仅仅对下一个组起作用，一个组为单个字符或者使用{}包裹起来的内容

例如：

$x^2$→x2\rightarrow x^2→x2

$10^10$会得到 101010^101010
要得到101010^{10}1010应该写成$10^{10}$。

  • 大括号可消除二义性（一种写法有两种解释，导致无法识别而出错）。

  • 注意一些写法上的区别：$x_i^2$展示为xi2x_i^2xi2​，而$x^{i_2}$展示为xi2x_{i^2}xi2​

根号、分数、括号、矢量、绝对值、范数

  1. 根号：通用表达方式为$\sqrt[a]{b}$→ba\rightarrow \sqrt[a]{b}→ab​
  • []内表示开a次方，省略则表示开平方
  • 如果被开方的是单个字符，{}可以被省略，如$\sqrt[a]b$→ba\rightarrow \sqrt[a]b→ab​
  1. 分式：分式有以下表示方法
  • 第一种使用$\frac {a}{b}$→ab\rightarrow \frac {a}{b}→ba​，其中a和b为单个字符时{}可以省略
  • 第二种使用$\over$来分割一个组的前后部分，$a+1 \over b+1$→a+1b+1\rightarrow {a+1 \over b+1}→b+1a+1​
  • 连分式：使用使用\cfrac代替\frac或者\over，两者效果对比如下：

\frac表示连分式：

x=a0+12a1+22a2+32a3+42a4+...x=a_0 + \frac{1^2}{a_ 1+\frac{2^2}{a_2+\frac{3^2}{a_3+ \frac{4^2}{a_4+...}}}} x=a0​+a1​+a2​+a3​+a4​+...42​32​22​12​

\cfrac表示连分式：

x=a0+12a1+22a2+32a3+42a4+...x=a_0 + \cfrac{1^2}{a_ 1+\cfrac{2^2}{a_2+\cfrac{3^2}{a_3+ \cfrac{4^2}{a_4+...}}}} x=a0​+a1​+a2​+a3​+a4​+...42​32​22​12​

  1. 括号：

  • 小括号和方括号：用()和[]即可。

  • **大括号：**使用时需要进行转义，因为{}本身已经用于分组。例：$\{a+b\}$→{a+b}\rightarrow\{a+b\}→{a+b}。也可使用\lbrace和\rbrace表示。

  • **尖括号：**使用\langle和\rangle分别表示左尖括号和右尖括号。例：⟨a+b⟩\langle{a+b}\rangle⟨a+b⟩

  • 向上取整：使用\lceil和\rceil表示。$\lceil x \rceil$→⌈x⌉\rightarrow \lceil x \rceil→⌈x⌉

  • **向下取整：**使用\lfloor和``\rfloor表示。$\lfloor x \rfloor$→⌊x⌋\rightarrow \lfloor x \rfloor→⌊x⌋

其中注意的是：原始括号不会随公式大小缩放，如果需要缩放需使用\left(...\right)。例如：$ \left( {1 \over 2}\over 2 \right)$→(122)\rightarrow \left( {1 \over 2}\over 2 \right)→(221​​)。原始括号效果为：(122)( { {1 \over 2}\over 2} )(221​​)

  • 绝对值：$\lvert...\rvert$。例如：∣x∣\lvert x \rvert∣x∣
  • 范数：$\lVert...\rVert$例如：∥x∥\lVert x \rVert∥x∥

省略号、空白间隔、分界符

  • **省略号：**有四种形式，分别为\dots、\cdots、\vdots、\ddots。

  • $\dots$ →…\rightarrow \dots→…，位置比较低，一般用于有下标的序列：

  • $\cdots$ →⋯\rightarrow \cdots→⋯，位置居中，一般用于正常序列：

  • $\vdots$ →⋮\rightarrow \vdots→⋮，竖直省略号，一般用于矩阵中。

  • $\ddots$→⋱\rightarrow \ddots→⋱，45∘45^{\circ}45∘方向省略号，一般使用于矩阵中。（注：45∘45^{\circ}45∘ 的表示方法为$45^{\circ}$）

  • 空白间隔：$1\quad1$→12\rightarrow 1\quad2→12 间隔距离为1em。1em （emphasize）= ? px 由该节点的父元素的font-size决定。

  • 上下限可用上标和下标表示：

$\sum_0^\infty$→∑0∞\rightarrow \sum_0^\infty→∑0∞​

  • 使用\to表示趋近于的箭头：$\lim_{x\to0}$→lim⁡x→0\rightarrow \lim_{x\to0}→limx→0​
  • 和、积、极限、积分等运算符用\sum, \prod, \lim, \int,这些公式在行内公式被压缩，以适应行高，可以通过\limits和\nolimits命令显示制动是否压缩。

$\int\limits_{-\infty}^{\infty} \frac {sinx}xdx$→∫−∞∞sinxxdx\rightarrow\int\limits_{-\infty}^{\infty} \frac {sinx}xdx→−∞∫∞​xsinx​dx
